## Что ищем

Конкретные цифры: стоимость токенов в рублях/долларах на типовую задачу SE,
данные FrugalGPT (до 98% экономии через каскадирование), данные CigaR
(73–96% через кэширование промптов), любые академические или отраслевые
модели TCO для LLM в разработке.

Особый интерес: есть ли работы, где измеряли стоимость N повторных вызовов
для одной и той же задачи? Это прямой gap для диссертации.
